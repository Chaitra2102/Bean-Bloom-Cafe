# Bean & Bloom Café

A modern café website built as a real business website project.

## Features

- Responsive café website
- Interactive menu
- Add to Cart functionality
- WhatsApp ordering
- Table reservations
- QR code based menu access
- Google Maps integration
- Contact information

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages
